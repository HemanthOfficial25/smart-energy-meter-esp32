# Smart Energy Meter using ESP32

## Overview
This project monitors electrical energy consumption using an ESP32 and ACS712 current sensor. The system measures current flow and helps users understand their power usage.

## Features
- Current Monitoring
- Energy Usage Tracking
- Real-Time Data Display
- Expandable to Cloud Dashboard

## Components
- ESP32
- ACS712 Current Sensor
- Breadboard
- Jumper Wires

## Working
The ACS712 sensor measures the current flowing through a load. The ESP32 reads the sensor output and displays the values through the serial monitor. The data can later be uploaded to cloud platforms for remote monitoring.
